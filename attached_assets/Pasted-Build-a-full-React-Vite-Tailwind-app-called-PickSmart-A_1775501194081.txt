NOVA VOICE FLOW
- when assignment loads, NOVA reads:
  Start aisle [start aisle]. End aisle [end aisle]. Total case count [total cases]. Total cube [total cube]. Total pallets [total pallets]. Goal time [goal time] minutes. To continue, say ready.
- then:
  Position Alpha. Get CHEP.
  Position Bravo pallet.

STRICT SLOT FLOW
- NOVA says: [aisle] [slot] check.
- selector must speak the correct 3-digit code
- if wrong, NOVA repeats heard input and says invalid
- system must never freeze
- loop until correct
- only after correct code: Confirmed. Grab [qty].
- then selector says ready and NOVA advances
