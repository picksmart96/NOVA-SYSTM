==================================================
1. WAKE / VOICE LAYER
==================================================
Build a voice layer for NOVA with these states:
- idle
- wake_listening
- active_listening
- thinking
- speaking
- stopped
- error

WAKE WORDS
- hey nova
- hola nova

STOP WORDS
- stop
- parar

BEHAVIOR
- user taps Start once to allow mic/audio
- then NOVA stays in wake_listening
- when user says hey nova / hola nova:
  NOVA says:
  - English: "Hi. I'm NOVA. How can I help you with selecting today?"
  - Spanish: "Hola. Soy NOVA. ¿Cómo puedo ayudarte con selección hoy?"
- after each NOVA answer, return to active listening
- if user says stop / parar:
  stop talking and return to wake_listening
- if browser voice fails, show fallback mode

FILES
- src/hooks/useVoiceEngine.js
- src/hooks/useNovaWakeWord.js
- src/components/nova/NovaVoiceStatus.jsx
